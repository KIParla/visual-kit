# KIParla Visual Kit

A reusable collection of vector assets developed within the KIParla research project.

## Contents

* Icons
* Illustrations
* Components

## Format

All assets are provided in SVG.

## Usage

You can:

* Use the assets in personal and research projects, no commercial use is allowed
* Modify and adapt them, preserving attribution
