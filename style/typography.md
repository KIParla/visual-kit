# Typography

## Primary Font

Lato

## Weights

* 400 — Regular (body text)
* 500 — Medium (UI elements)
* 600 — Semibold (subheadings)
* 700 — Bold (headings)

## Usage

* Headings → Bold / Semibold
* Body → Regular
* Interface → Medium

## Fallback

"Lato", system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
