| Primary 200 | #FFBBBB | UI subtle elements    |
| Primary 300 | #FF8888 | Decorative elements   |

## Notes

* The palette is based on KIParla's brand red (`#E52424`) as used consistently across all SVG assets
* Primary red is used for emphasis, interactions, and brand recognition
* Neutral colors dominate layout and readability
* Semantic colors are introduced for UI completeness
